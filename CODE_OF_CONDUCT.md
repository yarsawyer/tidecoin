# Code of Conduct

## Our Commitment

The Tidecoin community is committed to a collaborative, respectful, and
harassment-free environment for everyone, regardless of background or identity.

## Expected Behavior

- Be respectful and constructive in technical and social interactions.
- Assume good intent, ask clarifying questions, and focus on facts.
- Accept feedback gracefully and keep discussions on-topic.
- Prioritize project health, user safety, and responsible disclosure.

## Unacceptable Behavior

- Harassment, intimidation, threats, or personal attacks.
- Hate speech or discriminatory language.
- Repeated disruptive behavior, trolling, or bad-faith engagement.
- Publishing private information without explicit permission.

## Reporting

Report incidents to the maintainers via:

- Security channel: see [SECURITY.md](SECURITY.md)
- Repository issue tracker for non-sensitive moderation concerns:
  `https://github.com/tidecoin/tidecoin/issues`

If possible, include links, timestamps, and context so reports can be
reviewed quickly.

## Enforcement

Maintainers may take any action they consider appropriate, including:

- Warning and request to stop specific behavior.
- Temporary restrictions on participation.
- Removal of content, lock of discussions, or ban from project spaces.

Enforcement decisions are made to protect contributors and users, and to keep
project collaboration productive.

## Scope

This Code of Conduct applies in project spaces, including repository issues,
pull requests, discussions, and other official Tidecoin communication channels.
